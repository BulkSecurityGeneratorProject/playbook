La razón de este proyecto

La razón de ser de este proyecto es servir de banco de pruebas para distintas tecnología. Como dice el antiguo proverbio chino, "Lo que oigo, lo olvido; lo que veo, lo recuerdo; lo que hago, lo aprendo".

En principio servirá para practicar y aprender los distintos módulos de Spring, pero sin olvidar cualquier otro tipo de tecnología digno de experimentar como contenedores, tecnologías front-end, API Rest o CI, según las necesidades o el interés.
